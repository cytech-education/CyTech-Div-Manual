# 1on1

利用者との 1 対 1 面談（1on1）の実施ルールと面談内容、対応ポリシーをまとめています。

<template #common>

- 不可の場合も何かしらの方法で**ログをとる**こと
- ログは `07_1on1_Recording` に格納すること
  - 可能であればサイズ縮小・圧縮してください

</template>

## 対応ポリシー

| 項目 | 内容 |
|------|------|
| 対応曜日 | **土日対応可** |
| 対応時間 | **10:00〜19:00** |
| 振替対応 | 土日対応を行った場合、翌営業日の就業時間を削る（**早上がり対応**）ことで調整可 |

## Zoom 使い方メモ

- スケジュールリンクから入ると**許可待ち画面**になるので、Zoom アプリケーションの TOP ページから入ると良い
- 定期面談スケジュール（ICS リンク）で予定を取り込み可能
