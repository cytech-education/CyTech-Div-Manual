# テスト

利用者からのテスト提出時の対応フローをまとめています。

## テスト提出方法

- ユーザーからのテストは**チャット（Slack / Discord 等）経由**で送付される
- チャットにて内容を確認し、**クリア判定**を行う

## 合格時の対応フロー

1. **ユーザーへメッセージ送付**

   例：
   > テスト内容、確認しました！バッチリです💮 次のステップに進みましょう！

2. **管理画面にてステップ更新**
   - 対象ユーザーのステップ（フェーズ、進捗）を上位に更新
   - **更新漏れがないよう注意**

3. **IR の評価管理シートに記録**
   - 対象シート名：`CyTech 事業部_評価項目管理`
   - 対象ユーザーの該当行に、テストクリア情報を記載
   - 評価日・判定内容なども合わせて入力

## 注意事項

:::warning 対応時の注意
- **管理画面と評価シート、両方の更新を必ず行う**
- チャットでの確認後、できるだけ速やかに処理を完了する
- 判定に迷う場合は、**他の IR・管理者と相談して判断**する
:::
